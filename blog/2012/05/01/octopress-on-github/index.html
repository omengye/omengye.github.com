Octopress on Github

May 1st, 2012

对wordpress烦shi了,来试试octopress把. 优点(抄的)

1. 使用静态页面；
2. 不使用数据库；
3. 使用markdown标记语言编写文章；
4. 可以与git紧密集成,方便地进行博客的版本管理；
5. 可以于Github Pages集成,不需要单独的web hosting,只要你有github帐号即可。

首先的话就是构建git环境了,当然,文档中已经给的很清晰了,linux里配置git。
